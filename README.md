## 📥 Installation & Usage
1. Go to the [Releases](../../releases) page.  
2. Download the latest **`AS_Message_Sender.exe`** from the Assets section.  
3. Double-click the `.exe` file to start the application.  
4. No setup required – just **Run & Send Messages** ✅
